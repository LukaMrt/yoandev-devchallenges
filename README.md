# yoandev-devchallenges

Ce dépôt contient mes propositions de solutions aux défis hebdomadaires proposés sur [devchallenges.yoandev.co](https://devchallenges.yoandev.co/).

## Organisation du dépôt

- Un dossier par défi (une semaine = un dossier)
- À l'intérieur de chaque dossier de défi, un sous-dossier pour chaque solution proposée
